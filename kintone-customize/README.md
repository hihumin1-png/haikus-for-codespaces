# kintone-customize

This folder contains a base Node.js project for developing kintone customization JavaScript.

## Structure

- `package.json` - Node.js project configuration
- `src/app.js` - sample kintone customization script

## Usage

1. Install dependencies (if needed):
   ```bash
   cd kintone-customize
   npm install
   ```

2. Edit `src/app.js` and upload it to your kintone app as a JavaScript customization file.
